## 五、修复建议

### BUG-001 修复方案

**文件**：`frontend/src/components/TreeNode.tsx`

**当前代码**（应删除）：
```tsx
{isSelected && node.answer && (
  <div className="node-answer">
    {node.answer}
  </div>
)}
```

**删除理由**：
1. 回答内容应该在右侧主内容区展示，不应在左侧树内预览
2. 左侧树应该保持简洁，只显示问题标题
3. 当前实现导致左侧树过于拥挤，信息重复

**预期效果**：
- 左侧知识树只显示问题标题（带 truncate）
- 展开/折叠指示器正常
- 右侧主内容区显示完整的回答内容
